# Dev Nick - Portfolio Website

A modern, responsive portfolio website showcasing my work as a Web Developer and Software Engineer.

## About

This is my personal portfolio website built with HTML5, CSS3, JavaScript, and Bootstrap. It features:

- Responsive design that works on all devices
- Modern animations and transitions
- Skills showcase with interactive charts
- Portfolio projects section
- Contact form
- Resume download functionality

## Technologies Used

- HTML5
- CSS3 / SCSS
- JavaScript / jQuery
- Bootstrap
- Google Maps API
- Various JavaScript libraries (Stellar.js, Waypoints.js, Easy Pie Chart)

## Features

- **About Section**: Personal information and professional summary
- **Resume Section**: Work experience and education timeline
- **Services**: Overview of services offered
- **Skills**: Interactive skill charts
- **Portfolio**: Showcase of projects with GitHub links
- **Contact Form**: Email contact functionality
